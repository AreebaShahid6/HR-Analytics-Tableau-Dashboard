## 📌 Project Overview

This project presents an interactive **Human Resource Analytics Dashboard** built using **Tableau**. The dashboard transforms raw HR data into meaningful visual insights, helping organizations monitor workforce trends, employee demographics, salary distribution, and overall performance.

The goal is to support data-driven HR decision-making through intuitive visualizations and KPI tracking.
